What Is this?

This is a quickly built website made using bootstrap 4. Bootstrap was mainly used for the layout, custom CSS (technically SCSS, although unique SCSS features were barely used) was used for the stylings

Rules

Only use black red and white.

Use bootstrap 4, and freely edit the CSS

It must not resemble anything similar to the last 2 speed sites I made.

Fonts

The fonts that ive used are from font.googleapis.com. They are Amatic SC and Josefin Slab.

All the CSS (SCSS) used is as following

@mixin important-text{
 font-family: 'Amatic SC', cursive;
 color: rgb(255,100,100);
 font-weight: bold;
}
@mixin default-text{
  font-family: 'Josefin Slab', serif;
}
body{
  @include default-text;
  font-size: 30px;
  background: url('../../images/logo.png') no-repeat center center fixed;
  background-size: cover;
}
h1{
  @include important-text;
  font-size: 120px;
  padding: 70px 0;
}
h2{
  @include important-text;
  font-size: 60px;
  padding: 20px 0;
  transform: rotate(-0.002turn);
}
.bg-clear-dark{
  background-color: rgba(0,0,0,0.8)
}
pre{
  font-size: 12px;
}

Insight

It looks quite arty, altough there is no navigation. Perhaps the bottom images would be better as navigation after the content has been read.
